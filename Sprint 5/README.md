Sprint 5 - Storytelling With Tableau

In this project, I analyzed data to determine why customers are returning orders and how to reduce returns. I prepared several charts and graphs depicting the rate of return by location, timeframe, and customer, then created an interactive dashboard to monitor retail returns data over time. Lastly, I prepared and presented a Tableau story to summarize insights and provide relevant recommendations.
