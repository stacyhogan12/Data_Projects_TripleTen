Sprint 4 - Data Visualization With Tableau

In this project, I analyzed data from a retail establishment to determine profits and losses by product, then prepared visualizations to determine which products the store should stop selling and to identify which states and months the store should advertise in. Additionally, I analyzed the data to determine which products have the highest return rate and why.
